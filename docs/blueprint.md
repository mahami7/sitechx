# **App Name**: Smart Environment Portfolio

## Core Features:

- Company Introduction: Showcase the company's mission and core values.
- Services Overview: Provide a brief description of the key services offered.
- Contact Form Submission: Allow users to send contact messages which will be stored in Firestore.
- WhatsApp Integration: Enable click-to-chat via WhatsApp button.
- Mobile Responsiveness: Ensure the website is fully responsive on various devices.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to convey trust, stability, and innovation, reflecting a tech-focused look.
- Background color: Light Gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Electric Blue (#3B82F6) to highlight important calls-to-action and interactive elements.
- Font: 'Inter' sans-serif for headlines and body text to ensure a clean and modern user experience.
- Use minimalist icons related to smart technology, automation, security, and energy solutions.
- Ensure a clean, intuitive, and professional layout with clear sections and hierarchy.
- Implement subtle transitions and loading animations to enhance user experience.
